# **Asterisk Docker Setup**

## **Overview**

This repository contains a Dockerized Asterisk 18.22.0 setup with PJSIP integration.
It includes basic SIP configuration and dialplans for internal testing using clients like MicroSIP or Zoiper.

---

## **Commands**

### **1. Build and Start Asterisk**

```bash
docker compose up --build -d
```

---

### **2. Access Asterisk CLI (Direct)**

```bash
docker exec -it asterisk asterisk -rvvv
```

---

### **3. Stop and Remove Containers**

```bash
docker compose down
```

---

### **4. Access Container Shell**

```bash
docker exec -it asterisk bash
```

Then run inside the container:

```bash
asterisk -rvvv
```

---

## **Basic Info**

* **SIP Port:** 5070 (UDP/TCP)
* **Web Port:** 8088 (HTTP/WebSocket)
* **Configured Extensions:**

  * 1001 / 1001pass
  * 1002 / 1002pass

Use these with **MicroSIP** or **Zoiper** to register and test calls between extensions.
